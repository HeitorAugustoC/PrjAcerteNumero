# Instruções para a execução do projeto
> Executar o comando "npm install"
> Executar o comando "npx expo start"
> Ler o QR code através do aplicativo EXPO GO no celular
> Ou apertar a tecla W do teclado para iniciar a versão WEB
> Caso a versão WEB não funcione, execute o comando "npx expo install react-dom react-native-web" para instalar novamente as depêndencias para executar a versão web


Depêndencias usadas:
    "expo": "~54.0.34",
    "react": "19.1.0",
    "react-dom": "19.1.0",
    "react-native": "0.81.5",
    "react-native-web": "^0.21.0"
